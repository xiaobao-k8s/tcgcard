# 后续生成卡片规则 · 保持一致 + 合规

## 一、风格一致的关键

每次新增卡片，都固定以下视觉系统：

1. **圆形塑料旋风收藏卡**：不是方形卡、不是徽章、不是技能图标。
2. **2000 年代怀旧日式动画风**：粗线条、鲜艳配色、轻微 cel-shading。
3. **光栅/全息/塑料光泽**：要有卡片材质，但不能变成金属奖章。
4. **主体居中**：清晰正面或三分之二正面，轮廓要一眼看懂。
5. **无文字**：无文字、无数字、无 Logo、无字母。

## 二、合规的关键

不要只靠“原创设计”四个字。真正要合规，重点是避免“官方角色一一映射”。

### 推荐

- 原创生物名称：火尾幼蜥、赤冠烈蜥、炎翼龙兽。
- 原创结构：红晶短脊、熔岩翼缘、叶芽晶核、莲冠状花叶、贝壳状喷流鳍。
- 通用风格：2000 年代日式动画、圆形收藏卡、光栅质感。

### 避免

- 官方角色名、官方作品名、品牌名。
- “像某某角色”“类似某某最终进化”。
- 高度标志性的组合：特定颜色 + 特定器官 + 特定进化顺序完全一致。
- 复刻真实卡牌边框、编号、Logo、版权文字。

## 三、让进化差异明显的方法

### 1 阶：基础形态

- 关键词：小型、圆润、友善、基础属性特效。
- 主体占比：55%–60%。

### 2 阶：中阶形态

- 关键词：更高、更成熟、更锋利、更警觉。
- 主体占比：60%–65%。
- 常态卡不要做成攻击卡。

### 3 阶：最终形态

- 关键词：大型、宽轮廓、标志器官完全展开、王者气势。
- 主体占比：65%–70%。
- 必须换一个明显剪影，例如翼、冠、甲、喷流鳍、藤蔓群等。

## 四、Frame A / Frame B 差异

- Frame A：当前形态，正常姿态，特效温和或中等。
- Frame B：进化或攻击形态，体型更大，姿态更主动，特效更强，背景更深。

提示词里直接写：

```text
Frame B 必须比对应 Frame A 视觉存在感大 20%–35%，属性特效更强，背景更深，姿态更有攻击性；但主体轮廓必须清楚。
```

## 五、通用负面约束

```text
不要官方角色名，不要现有 IP 角色，不要品牌 Logo，不要文字、数字、字母、水印。不要金属徽章，不要游戏技能图标，不要 Q 版贴纸，不要写实 3D，不要复杂 UI，不要复刻真实商业卡牌边框。
```

## 六、推荐工作流

1. 每条进化链先生成 1 阶 Frame A。
2. 确认风格后，用同一套全局风格继续 1 阶 Frame B。
3. 中阶 Frame A 单独生成，明确“常态，不攻击”。
4. 最终形态先生成常态，再生成攻击帧。
5. 每次最多 1–3 张，避免批量过大导致风格漂移或失败率升高。
