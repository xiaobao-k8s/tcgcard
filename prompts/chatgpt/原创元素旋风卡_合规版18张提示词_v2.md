# 原创元素旋风卡 · 合规版提示词包 v2

> 适用目标：生成一组“原创宠物对战圆形旋风收藏卡”图片。  
> 本版本保留：圆形卡片、2000 年代怀旧日式动画风、鲜艳配色、粗线条、光栅/全息/塑料卡质感、Frame A / Frame B 差异。  
> 本版本移除：任何官方角色名、品牌名、Logo、编号文字、商业 IP 标志性组合与过度一一映射。

---

## 0. 合规原则

1. **不写官方 IP 名称**：不出现 Pokémon、Nintendo、Game Freak、Cheetos、奇多等品牌或作品名。
2. **不直接复刻官方角色**：不使用官方角色名，也不要求“像某某角色”。
3. **不做一一映射式外观**：保留“火/草/水 + 进化链 + 圆形卡片”的通用创意，但在体型、器官、颜色细节、背部结构、攻击方式上做原创化。
4. **不生成 Logo / 文字 / 数字**：所有图片都必须是无文字卡面插画。
5. **风格只引用通用时代感**：使用“2000 年代怀旧日式动画风、圆形旋风收藏卡、光栅卡质感”等通用审美，不要求复刻某个具体商业卡牌。
6. **边框原创**：边框简洁、有塑料卡片质感，避免复刻真实商品边框、符号、编号、版权信息。

## 2. Frame A / Frame B 区分规则

### Frame A：当前形态 / 常态卡

- 体型：正常占比，主体约占圆片内部 55%–65%。
- 姿态：稳定、警觉、站立、展示形态特征。
- 特效：中等或温和属性特效。
- 背景：标准属性渐变，干净、有光栅感。
- 气质：清楚展示“这个阶段长什么样”。

### Frame B：进化 / 攻击 / 更强状态

- 体型：比对应 Frame A 视觉存在感大 20%–35%；跨阶段最终形态可大 40% 左右。
- 姿态：更强、更主动，战斗姿态、蓄力、释放技能。
- 特效：属性特效明显增强，能量更亮、背景更深。
- 背景：更强烈属性色，允许全息光芒、能量爆发，但不要混乱。
- 气质：明确告诉观众“这是更强状态”。

## 3. 后续扩展卡片的一致性规则

### 3.1 固定不变的视觉参数

- 画幅：1:1，建议 1024×1024。
- 构图：圆形塑料旋风收藏卡居中。
- 主体：原创元素生物居中，清晰正面或三分之二正面。
- 描边：粗线条，2000 年代日式动画感。
- 上色：高饱和、轻 cel-shading。
- 材质：轻微光栅、全息、塑料光泽、轻 3D 厚度。
- 禁止：文字、数字、Logo、字母、水印、真实品牌边框。

### 3.2 每条进化链必须固定的变量

新增一条链时，先确定下面 6 个锚点：

| 锚点 | 要写清楚什么 | 示例 |
|---|---|---|
| 元素属性 | 火 / 草 / 水 / 雷 / 冰 / 岩 / 风等 | 火属性 |
| 主色 | 主体颜色 + 背景渐变 | 橙红主体，红橙背景 |
| 形态家族 | 生物大类，但不要使用官方角色名 | 火蜥、岩甲兽、风羽狐 |
| 阶段差异 | 小型 → 中型 → 大型的变化 | 小火尾 → 赤冠 → 炎翼 |
| 标志器官 | 原创识别点 | 晶石脊、月牙角、叶冠、贝壳鳍 |
| 属性特效 | 每阶段递增 | 小火星 → 火环 → 火翼爆燃 |

### 3.3 合规写法

推荐写：

```text
原创中型火属性爬行动物，橙色鳞片，背部有红色晶石状短脊，尾端有火焰。
```

避免写：

```text
像某某官方角色 / 某某宝可梦 / 某某最终进化 / 官方蓝色翅膀龙。
```

### 3.4 防跑偏负面约束

每张图最后可以加：

```text
不要做成金属徽章，不要做成游戏技能图标，不要 Q 版，不要贴纸感，不要写实 3D，不要复杂 UI，不要出现文字、数字、Logo、字母。
```

# 6. 批量生成建议

1. **不要一次生成太多**：优先 1 张 1 张生成，或每批 2–3 张。跨阶段、跨形态越多，失败率越高。
2. **先定风格锚点**：每条链先生成第一张常态卡，确认边框、光栅、上色风格后，再继续同链后续卡。
3. **Frame B 单独加重差异**：Frame B 的提示词必须明确“比 Frame A 更大、更强、特效更明显”。
4. **中阶常态不要写成攻击帧**：例如 xfd-005-frame-a 这类中阶常态，要用“稳定站立、警觉、自信”，不要“前扑、怒吼、火焰爆发”。
5. **最终形态避免官方剪影**：可以是龙兽、森兽、潮甲巨兽，但颜色、器官、攻击方式要原创化。
6. **失败后优先改主体描述**：如果被拒或效果偏官方 IP，先改“标志器官”和“配色组合”，不要只删除品牌词。
