# ChatGPT 生图反馈 - 问题与改进要求

> 基于上一轮生成的 18 张卡片预览图

---

##  主要问题

### 1. 进化差异不明显（最严重）

**问题**：同一进化链的卡片看起来几乎一样，没有体现进化变化。

**要求**：
- **小火龙链**：
  - 小火龙（frame-a）：小型橙色恐龙，尾巴火焰
  - 火恐龙（frame-b）：中型，更凶猛，火焰更旺 ← 应该明显更大更强
  - 火恐龙（frame-a）：中型姿态
  - 喷火龙（frame-b）：大型龙形，有翅膀 ← 应该完全不同

- **妙蛙种子链**：
  - 妙蛙种子（frame-a）：小型，背上种子球茎
  - 妙蛙草（frame-b）：中型，花苞绽放 ← 应该更大
  - 妙蛙草（frame-a）：中型姿态
  - 妙蛙花（frame-b）：大型，巨大红花 ← 应该完全不同

- **杰尼龟链**：
  - 杰尼龟（frame-a）：小型乌龟
  - 卡咪龟（frame-b）：中型，龟壳锋利 ← 应该更大
  - 卡咪龟（frame-a）：中型姿态
  - 水箭龟（frame-b）：大型，背上有两门水炮 ← 应该完全不同

### 2. 卡片风格不对

**问题**：图片像简单的游戏图标/贴纸，不是"2001 年奇多旋风卡风格"。

**要求**：
- ✅ 圆形卡片构图（直径约 4cm 感）
- ✅ 2000 年代怀旧日式动画风
- ✅ 鲜艳饱和配色，粗线条描边
- ✅ 光栅卡质感（轻微 3D 深度、光泽表面）
- ✅ 精灵正面朝向，动态但轮廓清晰
- ✅ 干净插图，不要任何文字
- ✅ 分辨率 1024×1024，精灵居中
- ❌ 不要 Q 版/简化风格
- ❌ 不要游戏图标风格

### 3. 没有体现光栅效果

**问题**：Frame A 和 Frame B 应该有不同的视觉效果。

**要求**：
- **Frame A（基础形态）**：正常姿态，温和特效
- **Frame B（进化/攻击形态）**：
  - 精灵尺寸更大（明显）
  - 特效更强烈（火焰更大、水滴更多、能量更明显）
  - 背景更强烈（颜色更深、光效更强）
  - 可以添加全息光芒效果（legendary 级别）

### 4. 属性特效缺失

**问题**：没有看到明显的属性特效。

**要求**：
- 🔥 **火属性**：火焰环绕、火星飘散、暖橙红色光芒
- 🌿 **草属性**：叶片飘落、藤蔓缠绕、绿色自然能量
- 💧 **水属性**：水滴飞溅、水波纹、浅蓝色清凉光芒

## ✅ 改进后的生成要求

### 每次生成 1 张图片（不要拼图）

**示例提示词**：
```
生成一张 2001 年奇多 Pokémon 旋风卡风格的圆形卡片插画。

精灵：小火龙（Charmander），一只橙色的小恐龙，尾巴末端有燃烧的火焰。
姿态：活泼的站立姿态，双手微微张开，尾巴火焰明亮，表情可爱。
属性特效：小火苗飘散在身体周围，暖橙色光芒。
背景：火属性色调渐变（橙色到红色），圆形卡片边框。
风格：2000 年代怀旧日式动画风，鲜艳饱和配色，粗线条描边，光栅卡质感。
分辨率：1024×1024，精灵居中，干净插图，不要任何文字。

输出文件名：xfd-004-frame-a.png
```

### Frame A vs Frame B 的区别

**Frame A（基础/当前形态）**：
- 精灵尺寸：正常
- 姿态：温和/正常
- 特效：基础属性特效
- 背景：标准属性色调

**Frame B（进化/攻击形态）**：
- 精灵尺寸：**明显更大**（比 Frame A 大 20-30%）
- 姿态：**更强/更凶猛**（战斗姿态、攻击姿态）
- 特效：**更强烈**（火焰更大、能量爆发、光效增强）
- 背景：**更强烈**（颜色更深、光效更强）

## 关键提醒

1. **每次只生成 1 张图片**，不要拼图
2. **Frame B 必须比 Frame A 明显更大、更强、特效更明显**
3. **进化后的精灵必须完全不同**（小火龙→火恐龙→喷火龙是三种不同的精灵）
4. **保持奇多旋风卡风格**（圆形、光栅质感、怀旧动画风）
5. **每张图都要有对应的属性特效**
